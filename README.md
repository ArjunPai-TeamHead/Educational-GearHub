*Educational-GearHub
**Educational-GearHub** is a web-based project developed fully in HTML. It is designed to provide educational resources, tools, or interactive content via an easy-to-access platform—all you need is a web browser.

## Features

- 100% HTML-powered (no backend/server required)
- Simple to deploy and host anywhere
- Cross-platform—works on any modern browser
- Easily customizable and extensible

## Getting Started

You can run the project locally by following these steps:

### 1. Clone the Repository

```bash
git clone https://github.com/ArjunPai-TeamHead/Educational-GearHub.git
cd Educational-GearHub
```

### 2. Open in Your Browser

Simply open the main HTML file (usually `index.html`) in your web browser:

- On most systems, you can double-click `index.html`, or
- Right-click and select "Open with" → your preferred web browser.

Alternatively, you can drag and drop the HTML file into your browser window.

### 3. (Optional) Use a Local Web Server

Certain features (like fetching resources with AJAX, or enhanced security) might require running a local server:

**Using Python 3:**
```bash
# From the project root directory
python3 -m http.server 8000
```
Then, open [http://localhost:8000](http://localhost:8000) in your browser.

**Using Live Server (VS Code Extension):**
1. [Install Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
2. Right-click `index.html` → "Open with Live Server"
